# Lavage Nova — Website (Demo)

Marketing site for **Lavage Nova**, mobile car wash in Agadir.

## Run

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Demo content

- Brand: Lavage Nova (navy `#0B1F3A` + cyan `#2BB3FF`)
- Phone / WhatsApp: `070 950 5971`
- Prices & testimonials are **demo placeholders**
- Creative section: **Mission Propre** mini-game (`#jeu`) → code `NOVA-PROPRE10`
- Before/after sliders with stock images (replace with real Agadir jobs)
